# Step 1: Zero-Shot Audio Classification (Web Console)

      1. Open the Hugging Face website
      2. Search for a Zero-Shot Audio Classification model or Space
      3. Open the model demo page
      4. Upload an audio file (speech, music, or noise)
      5. Enter candidate labels (e.g., speech, music, environmental noise)
      6. Click Run / Submit
      7. Observe predicted labels and confidence scores

# Step 2 : Automatic Speech Recognition (ASR) (Web Console)

      1. Search for an Automatic Speech Recognition model or Space
      2. Open the demo page
      3. Upload a speech audio file
      4. Click Transcribe / Run
      5. Observe the generated text transcription

# Step 3: Text-to-Speech (TTS) (Web Console)

      1. Search for a Text-to-Speech model or Space
      2. Open the demo interface
      3. Enter a text sentence
      4. Click Generate / Run
      5. Listen to or download the generated speech output
